## Usage

On a freshly installed [Anarchy Linux](https://anarchyinstaller.gitlab.io/)

Become root:
`sudo su` then:

```bash
pacman -Syyu git
git clone https://github.com/notional-labs/notional
cd notional
bash infrastructure/standup.bash
```
